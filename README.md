* 一般使用者
  * 首頁有兩個區塊
    * 公司區塊隨機顯示五間公司以及有一個公司列表的連結
    * 職缺區塊隨機顯示十個『開放』職缺以及一個職缺連結
  * 公司列表
    * 列出所有『開放』之公司
    * 點擊公司可進入公司詳細頁面
  * 職缺列表
    * 列出所有『開放』之職缺
    * 點擊職缺可以進入職缺詳細頁面
  * 公司詳細頁面
    * 公司資訊
    * 該公司的『開放』職缺
  * 職缺詳細頁面
    * 職缺資訊
    * 應徵按鈕
